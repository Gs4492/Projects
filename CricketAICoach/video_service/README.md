# Video Service

MVP service for club-level cricket video analysis.

## Current scope

- Upload a net-practice video
- Create a reusable session record per player
- Run first-pass structured analysis for batting or bowling
- Return coaching-ready JSON that can later be upgraded with CV models

## Planned later

- Ball-by-ball clip segmentation
- Pose estimation
- Shot-type classification
- Contact timing
- Bat-path and footwork scoring
