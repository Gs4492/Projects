# Phase 1 + 2 + 3 Frontend Tester

Open `index.html` in a browser (or serve folder with any static server).

Service defaults:
- player_service: http://127.0.0.1:8001
- match_service: http://127.0.0.1:8002
- analytics_service: http://127.0.0.1:8003
- ai_service: http://127.0.0.1:8004
- gateway: http://127.0.0.1:8000

Flow:
1. Create Player
2. Create Match
3. Load sample deliveries or paste your own JSON
4. Upload deliveries (bulk)
5. Fetch Phase 2 analytics
6. Fetch Phase 3 AI coaching
